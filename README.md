# Portfolio Website

Static personal website. Built with vanilla HTML, CSS, and JavaScript without external frameworks.

## Overview
The site serves as a central hub for my projects and contact information. It implements a custom theme switcher using CSS variables and DOM manipulation.

## Technical Details
* **Frontend:** HTML5, CSS3 (Grid/Flexbox), JavaScript (ES6).
* **Theme System:**
  * Default: Dark Mode.
  * Toggling: Button-triggered class/attribute switch.
  * Persistence: Uses browser `localStorage` to save user preference.
* **Compatibility:** Mobile-responsive layout.
